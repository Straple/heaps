# biheap
бинарная куча с минимумом
* empty O(1)
* top O(1)
* pop O(log2 n)
* push O(log2 n)

ТЕСТИРОВАНИЕ

test2 = рандомные доставать и забирать элементы. Забираем элементы с вероятностью 30%
